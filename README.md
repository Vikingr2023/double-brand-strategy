# 双品牌双赛道战略：从微舆到数瞰 DataKan + WeiSense

> 一个 AI 布道师的真实品牌战略决策记录 · 2026-07-01 拍板

## 🎯 核心决策

用**双品牌双赛道**策略，把一个舆情 SaaS 升级为两个独立品牌：
- **数瞰 DataKan** → 政企/中大型企业（高端企业 SaaS）
- **WeiSense** → 个人/创业团队（轻量现代 SaaS）

底层共用同一套数据中台（bettafish/insight/media/query），前端 UI/价格/CTA 完全不同。

## 📊 三段式战略叙事

### 为什么从"微舆"升级到"数瞰"？

| 旧名「微舆」 | 新名「数瞰 DataKan」 |
|---|---|
| 微博+舆情的拼接感，像功能词 | 「数」=数据，「氪」=俯瞰全局 |
| 底层是开源代码（BettaFish/Insight/Media/Query） | 不绑死舆情赛道，可扩展 AI 分析/出海/CRM/客服 |
| 国际化弱（Weiyu 拼音方案） | DataKan 5 字母好记·国际范 |
| Logo 难做（"微"字笔画复杂） | "数"+"氪"笔画有几何感，比"微"好做 100 倍 |

### 品牌延展性战略

> 微舆底层 = BettaFish/Insight/Media/Query 开源代码名 → CRM/客服/营销自动化场景不适配
>
> → 品牌升级为「数瞰 DataKan」= 品牌延展性成为品牌战略核心指标

**铁律：绝不混客、绝不互导流量、绝不共用 SaaS 后台。**

## 💰 漏斗式计费模型

老板点醒的"埋钩子"哲学：从轻量体验到长期订阅到企业定制。

```
¥13 体验（AI 减弱） → ¥299 完整版（多维度+排版） → ¥3,000 月监控（长期锁定） → 数瞰定制（销售经理对接）
```

### ¥13 体验版（AI 减弱）
- **快速给个轮廓**：声量+情感+Top10+7 天趋势
- **减弱什么**：无竞品对比、无处置建议、无危机预警、无专业排版、无后续设计
- 10 分钟出报告·一次一付·无需订阅

### ¥299 完整版（多维度+排版+后续设计）
- **多维度深度**：10 个关键词+30 天趋势+Top 20+竞品对比（3 个）+危机预警评估
- **专业排版**：设计师级排版+信息图表+配色体系
- **后续设计**：二次解读建议+报告结构化标签
- 一次一付·出 PDF+在线链接

### ¥3,000 月监控版（包月）
- **长期锁定**：不限报告+每日简报+5 分钟实时预警+API 接入+7 天数据保留
- 年付 9 折（¥32,400/年）

### 数瞰企业版/项目版（终极漏斗）
- **数瞰企业版**：¥3,000/月起，面向中大型企业
- **数瞰项目版**：定制服务，联系销售经理
- **数瞰标准报告**：¥999/次（首次企业注册免费）

## 💡 关键洞察

### 1. 品牌延展性 > 单一调性
「微舆」= 舆情监控单一赛道，**不绑死赛道**才有未来。「数瞰」= 数据洞察/AI 分析/CRM/客服/营销自动化**全场景品牌**。

### 2. 双品牌双赛道 > 单品牌多档
单品牌多档位（政企/小微/个人）会让品牌调性混乱。双品牌彻底隔离：数瞰 = 高端商务 / WeiSense = 轻量现代。

### 3. 漏斗式计费 > 平铺式
¥13 体验是钩子（AI 减弱版）→ ¥299 完整版是价值跃迁（多维度+排版）→ ¥3,000 月监控是长期锁定 → 数瞰定制是终极漏斗。

### 4. 主体隔离 > 借壳
数瞰放在鑫讯网络科技（912201026833802835，2010 年成立）旗下，**与数据洞察业务 100% 匹配**，而不是借壳金满堂文化传媒。
