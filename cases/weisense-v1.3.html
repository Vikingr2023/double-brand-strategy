¥13 体验版「AI 减弱」具体减弱什么？
¥13 体验版只给"轮廓"：声量总数+情感分布+Top 10 关键内容+7 天趋势图。不含：竞品对比、危机预警、AI 处置建议、专业排版、后续设计。简单说：知道"发生了什么"，但不知道"为什么"和"怎么办"。
¥299 完整版的「多维度+排版」具体包含什么？
¥299 完整版 = 体验版全部 + 10 个关键词 + 30 天趋势 + Top 20 内容 + 竞品对比（最多 3 个）+ 危机预警评估 + AI 处置建议 + PDF 专业排版 + 二次解读设计。简单说：知道"发生了什么、为什么、怎么办"，还能直接交给老板/客户。
¥3,000 月监控版适合谁？
需要长期监控品牌口碑/竞品动态的创业团队、MCN 工作室、PR 顾问、中小企业市场部。包含完整版全部功能 + 不限报告次数 + 每日简报 + 5 分钟实时预警 + API 接入 + 7 天数据保留。
三档都需要订阅吗？
¥13 和 ¥299 都是一次一付，无需订阅。¥3,000 月监控版是按月订阅，可随时取消。
WeiSense 和数瞰是什么关系？
两者都是鑫讯网络科技旗下舆情洞察产品，共享同一套数据底台。WeiSense 面向个人/小微，主打轻量、即时、便宜（¥13/¥299/¥3,000）；数瞰面向政企，主打严肃、专业、定制（¥999/次起 + 定制服务）。
