## 🎯 案例核心

### 数瞰 DataKan v4.1
- **定位**：政企/中大型企业高端 SaaS
- **主体**：鑫讯网络科技（统一社会信用代码 912201026833802835）
- **域名**：datakan.cn
- **核心模块**：
  - 全网报告样章（声量+情感+Top内容+趋势+竞品+危机+处置）
  - 企业注册表单（含信用代码校验+邮箱/手机双通道验证）
  - 5 分钟报告交付流程
  - 4 档套餐（首次免费 + ¥999/次 + ¥3,000/月 + ¥10,000/项目）

### WeiSense v1.3
- **定位**：个人/创业团队轻量现代 SaaS
- **主体**：鑫讯网络科技（同上）
- **域名**：weisense.cn
- **核心模块**：
  - 漏斗式三档对比（¥13 体验 → ¥299 完整版 → ¥3,000 月监控）
  - ¥13 体验版（AI 减弱结果 + 简单版报告）
  - ¥299 完整版（多维度 + 专业排版 + 后续设计）
  - ¥3,000 月监控版（包月 + 每日简报 + 实时预警 + API 接入）
  - 数瞰企业版终极漏斗

## 📊 关键设计决策

1. **品牌色严格隔离**：
   - 数瞰：深蓝 #0A1929 + 科技蓝 #1EB8FF + 数据绿 #00E58B + 高级灰 #9AA4B2（设计稿原值 · 暖橙 #FF6B35 是 RAS AI KHAIMAH 户外品牌主色）
   - WeiSense：清新蓝 #5B7CFA + 翠绿 #48BB78（轻量现代）

2. **漏斗式计费**：
   - ¥13 钩子（AI 减弱）→ ¥299 价值跃迁（多维度+排版）→ ¥3,000 月监控（长期锁定）→ 数瞰定制（终极漏斗）

3. **主体归属**：
   - 鑫讯网络科技 = 数据/AI/网络技术（与数瞰数据洞察业务 100% 匹配）
   - 不是借壳金满堂文化传媒

4. **技术架构**：
   - 韩国首尔 177 服务器（43.131.242.177）
   - Nginx 1.26.3 + Let's Encrypt SSL（90 天自动续期）
   - 同一数据底台（bettafish/insight/media/query）
